# Privacy

This site is a personal blog. It does not use cookies, tracking pixels, or advertising networks.

## What data is collected

**Server logs**: Standard HTTP request logs (IP address, user agent, timestamp) are retained by the hosting provider (Vercel) for operational purposes. These are not used for analytics or profiling.

**Comments**: Blog post comments are powered by [Giscus](https://giscus.app), which uses GitHub Discussions as a backend. When you post a comment, you authenticate directly with GitHub. No credentials pass through this site. GitHub's own [privacy policy](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) applies.

**Fonts**: Fonts are self-hosted. No requests are made to Google Fonts or other external font services.

## Third-party services

| Service | Purpose | Data shared |
|---------|---------|-------------|
| Vercel | Hosting | Standard HTTP request data |
| Giscus / GitHub | Comments | GitHub account (only if you comment) |
| Cloudflare | DNS | Standard HTTP request data |

## Your rights

Under GDPR, you have the right to access, rectify, or request deletion of any personal data. Since this site collects minimal data, there is likely nothing stored about you. If you have questions, reach out at [privacy@akj.io](mailto:privacy@akj.io).
